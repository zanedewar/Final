Inital Portfolio/ Resume

Isabella Mankis Portfolio

My first portfolio in an HTML format was, as anyone can plainly see, incredibly basic. This project is a great way to see how much I have learned over the course of one summer quarter. Although there is nothing wrong with this project in the sense that I could use this as a portfolio, it is simple compared to what I am now capable of. If I were to do this project again, I would make many changes, but, I am still proud of the work I created when I knew nothing about HTML formatting.

Madrid, Spain

The basics for the new Madrid tourist

My Madrid website was a true turning point in my learning. If you were to look at the website, you would be able to see the slideshow created at the top of the page. At this point in my learning, I was able to expand into other areas of HTML that were of interest to myself. The assignment used to create this website called for 4 pictures of the attractions I would describe on the page. I had decided that it was worth it to me to figure out how to create a slideshow because I really like websites that have slideshows. It was very challenging but worth it.

Wizard Of Oz Chapters 1-3

Chapters 1-3 of the Wizard of Oz

I appreciated this assignment as this, again, presented a new challenge. The assignment was all about CSS and had to include pictures. I decided to try to be inventive with the assignment and use the images as the background. I have seen this before with websites about books or stories and it seems to work well within those sites. I believe my mistake with this was not making the pictures faded enough as they can get in the way of what's being read. Other than that, I'm very happy with the site I created.

The Optimist: the daily news with a positive spin

The Optimist News

The Optimist is a news subscription that provides good news to anyone that signs up. On this website, you can see that I have created a form for user's to fill out before moving on to the next page. Everything within this site was created by myself from the code, to the logo, and the mock-up. I'm truly proud of this project as it has been my best yet. There are still things I would like to change but don't necessarily know how. Overall, I'm happy with the work I put forward as I know it's the best I could have done.
